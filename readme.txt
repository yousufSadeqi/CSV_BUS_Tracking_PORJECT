README – Note concernant le projet

Lors du développement du projet, j’ai rencontré des problèmes de configuration avec Code::Blocks, indépendants du code lui-même. Le projet Code::Blocks peut donc ne pas fonctionner correctement.

Afin d’éviter tout blocage lors de la correction, j’ai fourni le code sous deux organisations différentes :

une version avec les dossiers main_programme et complexite,

une version alternative sans projet Code::Blocks.

Le code a été entièrement vérifié et fonctionne correctement lorsqu’il est compilé sans fichier projet.

Les deux versions sont fournies par précaution.

Merci pour votre compréhension.
